# Bushido Hubit - ランディングページ

武士道と習慣をテーマにした、侘び寂びの美学を取り入れたモダンなランディングページです。

## 技術スタック

- **React 18** - UIライブラリ
- **Vite** - ビルドツール
- **Tailwind CSS** - スタイリング
- **Framer Motion** - アニメーション
- **Lucide React** - アイコン

## セットアップ

### 依存関係のインストール

```bash
npm install
```

### ロゴ画像の配置

プロジェクトの `public` フォルダに `logo.png` を配置してください。

- 推奨サイズ: 幅200-400px、高さはアスペクト比に応じて
- 形式: PNG（透明背景推奨）
- パス: `public/logo.png`

ロゴ画像が存在しない場合は、テキストロゴ「W/T」が表示されます。

### 開発サーバーの起動

```bash
npm run dev
```

ブラウザで `http://localhost:5173` を開いてください。

#### 同じネットワーク内の他のデバイスからアクセス

開発サーバー起動時に表示されるローカルIPアドレス（例: `http://192.168.1.100:5173`）を使用して、同じWi-Fiネットワークに接続されているスマートフォンやタブレットからアクセスできます。

**注意**: ファイアウォールの設定によっては、外部からのアクセスがブロックされる場合があります。その場合は、ファイアウォールの設定でポート5173を許可してください。

### ビルド

```bash
npm run build
```

### プレビュー

```bash
npm run preview
```

## デザインコンセプト

- **侘び寂びの美学**: 日本の伝統的な美学をデジタルデザインに取り入れています
- **ミニマリズム**: 無駄を削ぎ落とし、本質的な美しさを追求
- **静寂の美**: アニメーションはゆっくりと有機的で、落ち着いた印象を与えます
- **レスポンシブデザイン**: スマホからデスクトップまで対応
